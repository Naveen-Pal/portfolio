## Project Overview

OWASP Nest serves as a centralized gateway to the OWASP (Open Web Application Security Project) organization's resources. The project aims to simplify navigation and improve accessibility to OWASP's vast security knowledge base.

## Technical Stack

The application is built using modern web technologies:
- Frontend: React
- Backend: Django
- Focus on security best practices
